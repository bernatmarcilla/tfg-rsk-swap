# Usage

## Install Dependencies

```shell
npm install
```

## Run Development Environment

```shell
npm run dev
```

### App running at:
  - Local:   http://localhost:8080/
  - Network: http://192.168.1.70:8080/


## File Structure
Within the download you'll find the following directories and files:

```
|-- Vue Black Dashboard

```
